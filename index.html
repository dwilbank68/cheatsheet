Ruby Loop

Ruby has additional looping structures (apart from the common For..in and While loops) that are specific in use and aim to be a more semantic approach to a problem. Keep in mind that these can be replicated with the traditional For..in and While loops!

Each Loop

Executes code once for each element in a collection.

Times Loop

Executes code a given number of times. A Times Loop must be used with a variable containing a number or a number itself.

Until Loop

Executes code until a condition is true. This is the same as a negated while loop. ex: !(while)

Loop Loop

Executes code in an infinite loop.
